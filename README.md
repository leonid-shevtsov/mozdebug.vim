# MozDebug.vim - a frontend debugger for Vim

## Installation

1. Install [MozRepl](https://addons.mozilla.org/en-US/firefox/addon/mozrepl/) into Firefox (it's zero-config in most cases).
2. Install this plugin into Vim


## Usage

`:help mozdebug`
